Curator Features

Everything you as curator need to create effective learning pathways

Pathway Management

Create, edit, and organize learning pathways with multiple steps and detailed progression.

Easy Bookmarking

Get links into your pathways with easy manual entry, a bookmarklet, and more coming soon!

Comprehensive Export Options

Export pathways in various formats including HTML, RSS, CSV, and JSON for sharing.

Smart Import Options

Import pathways from JSON with a full conflict "diff" resolver. Collaborate even without GitHub.

Link Auditing

Verify the status of links in your pathways to ensure all resources are accessible.

GitHub Integration

Save and share pathways via GitHub repositories for collaboration, backup, and easy hosting.

Learner Features

Your learners get a curated pathway of links to follow.

Objectives

Each step along the pathway can have a clear, stated objective that aligns with the overall purpose.

Link Tracking

Your learners can track which links they've already visited and see a progress bar of their journey.

Pause & Reflect

After each step, you can have optional questions to provoke your learners to consider more.

Getting Started

No installation required - just visit and start creating!

Ready to create your first learning pathway?

PathCurator is 100% free and runs entirely in your browser. Your data is stored locally and can be synced with GitHub for backup, collaboration, and sharing. There are no accounts required to use it (but for a GitHub account if you use that feature), no ads and no tracking of any kind.

Launch PathCurator

How It Works

1
Create a Pathway

Click "New Pathway" to start creating your learning path

2
Add Steps

Use the "Add Step" button to create new steps in your pathway

3
Add Bookmarks

Within each step, add bookmarks to relevant resources

4
Export & Share

Export as HTML, CSV, or JSON files for sharing

Free, Open Source Software

PathCurator is fully open source, "local-first" and is even easily self-hostable.

Check it out on GitHub

Host a copy of the app yourself using any simple web server (including running it locally using the provided Python dev server).
